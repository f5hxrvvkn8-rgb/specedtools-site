# specedtools-site

Static site for [specedtools.org](https://specedtools.org) — hosts privacy policy and terms of service for the SpecEdTools iOS app.

## Pages

- `/` – landing page
- `/privacy.html` – Privacy Policy (English)
- `/terms.html` – Terms of Service (English)
- `/integritet.html` – Integritetspolicy (svenska)
- `/villkor.html` – Användarvillkor (svenska)

## Hosting

Served via GitHub Pages with the custom domain `specedtools.org` (see `CNAME`).

## Editing

Edit HTML files directly. Push to `main` and GitHub Pages redeploys within a minute.
